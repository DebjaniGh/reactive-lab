📊 scan() — The Accumulator

scan() remembers the previous value and accumulates over time.

scan() works like a scoreboard — it receives each new value, combines it with the previous result, and emits the running total. Unlike reduce(), which waits until the stream ends, scan() emits after every single value.

Real-world examples:

  • 🔢 Notification Badge — Each new notification increments a counter. scan() keeps the running count so the badge always shows the current total.
  • 🛒 Shopping Cart — Each "Add to Cart" click appends an item. scan() accumulates the items array and recomputes the total price after every addition.
  • 🔄 App State (NgRx) — Each user action dispatches an event. scan() applies a reducer function to produce the new state. This is literally the engine behind NgRx and Redux.
