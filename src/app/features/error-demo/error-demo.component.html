🛡️ Error Handling — catchError, retry, throwError

Same error. Three different strategies. Watch which streams survive.

In RxJS, an unhandled error terminates the stream permanently. WHERE you place catchError() determines whether the stream survives. Inside switchMap() — the stream lives. Outside switchMap() — the stream dies.
